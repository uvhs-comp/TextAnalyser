    A text Analyser made using Python 3 and
    Flask. Simple to use, just insert the url
    or directory of the file that you want to
    be analysed and press enter. Or if you'd
    prefer you can copy the text in to the box
    box for the same effect. The analyser return
    a wide array of results including uses of
    punctuation, words and sentences, and other
    features such as reading age and alliteration.
